# 라이나 통합건강1 금시상 콜리스트 자동생성기

엑셀 로우파일을 업로드하면 아래 기준으로 콜리스트 엑셀을 자동 생성합니다.

## 1차 버전 반영 기준

- 대상상품
  - 새로담는건강보험
  - 새로담는간편건강보험
  - 새로담는건강보험플러스
- 반영시책
  - 통합건강1 13회차 금시상 구간
- 인정기준
  - 기본값: 계약상태 = 유지, 납입상태 = 정상
  - 화면에서 체크박스를 해제하면 전체 계약 포함 가능
  - 건강보험 건당 30만원 한도
- 구간
  - 5만원 이상: 60만원
  - 10만원 이상: 140만원
  - 20만원 이상: 320만원
  - 30만원 이상: 540만원
  - 50만원 이상: 1,000만원

## 로컬 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 비밀번호 설정

로컬 테스트 시 `.streamlit/secrets.toml.example` 파일을 복사해서 `.streamlit/secrets.toml`로 이름을 바꾼 뒤 비밀번호를 입력하세요.

```toml
APP_PASSWORD = "팀원에게_공유할_비밀번호"
```

비밀번호를 설정하지 않으면 앱은 실행되지만, 배포 전에는 반드시 Streamlit Secrets에 `APP_PASSWORD`를 설정하는 것을 권장합니다.

## Streamlit Community Cloud 배포 요약

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 업로드합니다.
3. Streamlit Community Cloud에서 저장소, 브랜치, `app.py`를 선택해 배포합니다.
4. App settings > Secrets에 아래 값을 등록합니다.

```toml
APP_PASSWORD = "팀원에게_공유할_비밀번호"
```

5. 생성된 `.streamlit.app` 주소를 팀원에게 공유합니다.

## 개인정보 주의

업로드된 파일은 앱 코드에서 별도로 저장하지 않고, 메모리에서 처리 후 결과 파일만 다운로드하게 구성했습니다. 단, 회사 정책상 고객명/계약번호 등 개인정보가 포함된 파일을 외부 서비스에 업로드해도 되는지는 내부 기준을 먼저 확인하세요.
